Two ways to get games in your calendar — and why one is better

📡 Option A — Subscribe to .ics URL Current
✅Games appear on your calendar

✅Works on iPhone via iCloud

❌iPhone Focus modes cannot trigger from these events

📅 Option B — Write as Real Events (AppleScript) Proposed
✅Games appear on your calendar
✅Auto-updates every 6 hours via LaunchAgent
✅Syncs to iPhone via iCloud like any regular event
✅You are marked as BUSY during every game
✅iPhone Focus modes work — e.g. silence notifications during games
✅Events are editable — add notes, change alerts, RSVP
✅Shows as busy in shared calendars & Find a Time
💡 Option B keeps the same automation — the script still runs every 6 hours, fetches CBF + ESPN, and updates the events. It just writes them as real calendar entries on a dedicated "🦊 Cruzeiro" calendar you own, instead of a read-only subscription.

How to set it up (one time)
  1. iPhone → Settings → Focus
  2. Tap + → create "Game Time" focus
  3. Under Turn On Automatically → Add Schedule
  4. Choose "When a Calendar Event Matches"
  5. Filter by calendar: 🦊 Cruzeiro
  6. Done — it activates itself every game night
⚠️ This only works with real events (Option B). Subscribed calendars are invisible to Focus mode triggers.

What the script does each run

🗑️Deletes all existing events on the "🦊 Cruzeiro" calendar (so stale games don't pile up)
➕Re-adds every upcoming fixture with up-to-date times from CBF + ESPN
🕐TBD games become all-day events — promoted to timed events once CBF confirms
📲iCloud picks up the change and pushes to iPhone within minutes
🔒 Everything stays 100% local. No cloud service, no API keys for calendar access. The Python script talks to Calendar.app directly via AppleScript — the same way you'd drag an event yourself.
